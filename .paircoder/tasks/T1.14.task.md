# Objective

Complete all SEO and technical requirements across every page. Run PageSpeed and resolve issues before DNS cutover (T1.15).

**Depends on:** T1.4–T1.12 (all pages must exist)

# Implementation Plan

## 1. Meta tags audit — every page must have:
- Unique title tag (60 chars max) — values from spec Section 3.4
- Unique meta description (155 chars max) — values from spec Section 3.4
- Canonical tag pointing to the clean URL
- OG title, OG description, OG image tags

Check all 9 pages against spec values — fill any gaps left from page builds.

## 2. Schema JSON-LD
Add to `index.html` (homepage) AND `about/index.html`:
```json
{
  "@context": "https://schema.org",
  "@type": ["Person", "LocalBusiness"],
  "name": "Carol Merlo",
  "jobTitle": "Transformational Coach, Licensed Spiritual Practitioner",
  "description": "Carol Merlo, M.Ed. — spiritual and business coaching for conscious entrepreneurs.",
  "url": "https://carolmerlo.com",
  "sameAs": [
    "https://facebook.com/[carol-page]",
    "https://linkedin.com/in/[carol-profile]",
    "https://youtube.com/[carol-channel]",
    "https://instagram.com/[carol-handle]"
  ]
}
```
(Fill actual profile URLs when Carol provides them)

## 3. sitemap.xml
Generate manually or use a tool — include all pages:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url><loc>https://carolmerlo.com/</loc></url>
  <url><loc>https://carolmerlo.com/work-with-me/</loc></url>
  <url><loc>https://carolmerlo.com/business-coaching/</loc></url>
  <url><loc>https://carolmerlo.com/spiritual-coaching/</loc></url>
  <url><loc>https://carolmerlo.com/books/</loc></url>
  <url><loc>https://carolmerlo.com/blog/</loc></url>
  <url><loc>https://carolmerlo.com/about/</loc></url>
  <url><loc>https://carolmerlo.com/pricing/</loc></url>
  <url><loc>https://carolmerlo.com/contact/</loc></url>
</urlset>
```
Do NOT include `/admin` in sitemap.

## 4. robots.txt
Create at root — exactly per spec:
```
User-agent: *
Disallow: /admin/
Sitemap: https://carolmerlo.com/sitemap.xml
```
**Important:** Contact page was blocked in old Weebly robots.txt — do NOT block /contact/ here.

## 5. Google Analytics 4
- Add GA4 tracking snippet to every page's `<head>` (or load via `main.js` with defer)
- Use existing GA4 property if one exists; if not, create new property for carolmerlo.com
- Document the GA4 Measurement ID in a comment in main.js

## 6. _redirects file
Confirm `_redirects` (from T1.2) is correct — verify no typos, correct 301 codes.

## 7. Canonical domain
- Configure in Netlify dashboard: Site Settings → Domain → Force HTTPS + redirect www to non-www
- This is separate from the `_redirects` file

## 8. PageSpeed Insights
- Run Google PageSpeed Insights on the Netlify preview URL for the homepage
- Target: 90+ on both mobile and desktop
- Common fixes: convert images to WebP (done), add `width`/`height` attributes to all images (prevents CLS), confirm no render-blocking scripts
- Resolve any issues before DNS cutover

- [ ] Person + LocalBusiness schema JSON-LD on homepage and About page
- [ ] sitemap.xml includes all 9 content pages, excludes /admin
