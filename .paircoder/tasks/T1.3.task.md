# Objective

Build the four global components that every page depends on. These must be finished before any page content work (T1.4+). Extract as reusable HTML partials or CSS component classes.

**Depends on:** T1.2

# Implementation Plan

## 1. Navigation
- Logo: "Carol Merlo" as text in `--font-heading` / `--teal` color (or SVG if available)
- Main nav links: Home | Work With Me (dropdown) | Books | Blog | About | Contact
- Work With Me dropdown: two sub-links — Spiritual Coaching (`/spiritual-coaching`) | Business Coaching (`/business-coaching`)
- Mobile hamburger: smooth open/close, no flicker — JS in main.js
- Active state: highlight current page link
- Fixed or sticky position — choose and document the decision

## 2. Footer
- Nav links repeated: Home | Work With Me | Books | Blog | About | Contact
- Social icon links (use SVG icons or Unicode placeholder): Facebook | LinkedIn | YouTube | Instagram
- Copyright line: "© [year] Carol Merlo. All rights reserved."
- Background: `var(--teal-dark)`, text `var(--white)`

## 3. CTA Button component (CSS classes)
- `.btn-primary`: background `var(--teal)`, text `var(--white)`, border-radius `var(--border-radius)`, hover darkens to `var(--teal-dark)`
- `.btn-secondary`: transparent background, border `2px solid var(--gold)`, text `var(--gold)`, hover fills with gold, text white
- Both: standard padding, cursor pointer, no underline
- Both must work as `<a>` tags (not `<button>`) since all CTAs are Calendly links

## 4. Testimonial Card component (CSS class)
- `.testimonial-card`: quote text in italic, attribution name in `--text-mid`, subtle `var(--teal-light)` background, padding, optional left border in `--gold`
- Must be reusable across: Homepage, About, Spiritual Coaching

## Integration
- Include nav and footer HTML in every page shell created in T1.2
- Document component usage with a comment in main.css

# Acceptance Criteria

- [x] Navigation renders on all pages with correct links and dropdown
- [x] Hamburger menu opens/closes smoothly on mobile (test at 480px)

- [x] Footer appears on all pages with social links and copyright
- [x] `.btn-primary` and `.btn-secondary` both render correctly and hover state works
- [x] `.testimonial-card` renders with correct teal-light background
- [x] All components tested at 480px, 768px, and 1100px+ widths
