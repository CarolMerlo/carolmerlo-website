# Objective

Add a full-width banner image with centered text overlay as the first section of the homepage, above all existing content. Nothing is deleted or moved — this is purely additive.

# Prerequisites

- Carol must add `beautiful-background.jpg` to `assets/images/` before this task starts
- Compress image to under 300KB if needed (use squoosh.app)

# Implementation Plan

1. Add `.hero-banner` as the first `<section>` inside `<main>` in `index.html`:
```html
<section class="hero-banner">
  <div class="hero-banner-content">
    <h2 class="hero-banner-text">Real Change Begins Within</h2>
  </div>
</section>
```

2. Add to `pages.css`:
```css
.hero-banner {
  background-image: url('/assets/images/beautiful-background.jpg');
  background-size: cover;
  background-position: center;
  min-height: 420px;
  display: flex;
  align-items: center;
  justify-content: center;
}
.hero-banner-content {
  text-align: center;
  padding: var(--space-md);
}
.hero-banner-text {
  font-family: var(--font-heading);
  font-size: clamp(2rem, 5vw, 3.5rem);
  color: var(--white);
  text-shadow: 0 2px 8px rgba(0, 0, 0, 0.5);
  font-weight: normal;
  letter-spacing: 0.02em;
}
@media (max-width: 600px) {
  .hero-banner { min-height: 260px; }
}
```

3. Run `npm run lint` — fix any errors before committing

# Acceptance Criteria

- [ ] Banner is the first visible element on the homepage above all existing sections
- [ ] Text "Real Change Begins Within" is legible on the photo at all screen sizes
- [ ] All existing homepage sections are unchanged below the banner
- [ ] Image is fully responsive — no horizontal scroll, covers full width on all screen sizes
- [ ] PageSpeed mobile score does not drop below 85
- [ ] HTML and CSS linters pass

- Open homepage — banner is the first thing visible
- Resize to 375px width — text readable, image covers full width
