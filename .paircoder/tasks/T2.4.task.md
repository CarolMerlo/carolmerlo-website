# Objective

Fix the GA4 property configuration (wrong domain, missing business info) and add conversion event tracking for the actions that matter most to Carol's business goals.

# Implementation Plan

## Part A — Dashboard fixes (manual, no code — Carol or son)

Steps inside Google Analytics (analytics.google.com):

1. **Fix data stream domain:**
   Admin → Property → Data Streams → click the stream → change URL from `carolmerlo.net` to `carolmerlo.com` → Save

2. **Fill in business details:**
   Admin → Property → Property details → Industry: "Arts & Entertainment" or "Business & Industrial" → Business size: Small → Save

3. **Link Search Console:**
   Admin → Property → Search Console Links → Add link → select carolmerlo.com Search Console property

4. **Enable Enhanced Measurement:**
   Admin → Property → Data Streams → click stream → Enhanced measurement → toggle On (tracks scrolls, outbound clicks, file downloads automatically)

5. **Mark conversions:**
   After code is deployed and events are firing — Admin → Conversions → mark `calendly_click` and `contact_form_submit` as conversions

## Part B — Conversion event tracking (code changes to `main.js`)

Add inside `DOMContentLoaded`:

```javascript
/* GA4 conversion tracking */
// Calendly CTA clicks (all pages)
document.querySelectorAll('a[href*="calendly.com"]').forEach(function (el) {
  el.addEventListener('click', function () {
    gtag('event', 'calendly_click', {
      link_text: el.textContent.trim()
    });
  });
});

// Contact form submission
var contactForm = document.querySelector('form[name="contact"]');
if (contactForm) {
  contactForm.addEventListener('submit', function () {
    gtag('event', 'contact_form_submit');
  });
}

// Amazon book link clicks
document.querySelectorAll('a[href*="amazon.com"]').forEach(function (el) {
  el.addEventListener('click', function () {
    gtag('event', 'book_link_click', {
      link_text: el.textContent.trim()
    });
  });
});

// Popup subscribe (fired from T2.3 Mailchimp form submit — wire after Mailchimp embed is in place)
```

Note: `video_play` event is handled in T2.2.
