# Objective

Add 301 redirects to `netlify.toml` for all old Weebly `.html` URLs so Google does not hit 404s after DNS cutover. This is required before going live.

# Implementation Plan

- Open `netlify.toml` in project root
- Append the following `[[redirects]]` blocks after the existing `[[headers]]` blocks:
