# Objective

Build the Contact page (`contact/index.html`). Primary CTA is the Calendly Insight Call booking. Contact form is secondary. Social links included.

**Depends on:** T1.3

# Implementation Plan

## Layout
1. **Headline** — *Let's Connect*
2. **Primary CTA** — "Schedule a Complimentary 20-Minute Insight Call" — `.btn-primary` → Calendly URL. This is the FIRST thing on the page — leads with booking, not form.
3. **Contact form** (secondary) — Netlify Forms:
   ```html
   <form name="contact" method="POST" data-netlify="true">
     <input type="text" name="name" placeholder="Your name" required>
     <input type="email" name="email" placeholder="Your email" required>
     <textarea name="message" placeholder="Your message" required></textarea>
     <button type="submit">Send Message</button>
   </form>
   ```
   - `data-netlify="true"` attribute enables Netlify form handling — no server needed
   - Add hidden honeypot: `<input type="hidden" name="bot-field">`
4. **Social links** — icon links (use SVG or Unicode placeholder):
   - Facebook | LinkedIn | YouTube | Instagram

## Netlify Forms
- Form submissions go to Netlify dashboard — Carol notified by email
- No additional backend required

## SEO
```
Title: Contact Carol Merlo | Schedule a Complimentary Insight Call
Meta: Schedule a complimentary 20-minute insight call with Carol Merlo, or send a message. Spiritual and business coaching for conscious entrepreneurs.
```
- Canonical: `https://carolmerlo.com/contact/`
- robots.txt must NOT block this page (spec notes current Weebly robots.txt blocks contact — fix in T1.14)
- OG tags
