# Objective

Execute the DNS cutover from Weebly to Netlify. This is the point of no return for Weebly — keep Weebly live until Carol has approved every page on the Netlify preview URL. **Do not cancel Weebly until the new site is confirmed working on the live domain.**

**Depends on:** T1.13 (Decap CMS launch gate passed), T1.14 (PageSpeed 90+ achieved)

# Implementation Plan

## Pre-Cutover Gate (must pass before touching DNS)
1. Carol reviews EVERY page on the Netlify preview URL (not DNS-pointed yet)
2. Carol provides written approval (email or message) — document this
3. All items checked: every page, every link, every form, every CTA button
4. PageSpeed 90+ confirmed (from T1.14)
5. Decap CMS launch gate passed (from T1.13)

## DNS Cutover Steps
1. Log into domain registrar (carolmerlo.com is registered outside Weebly)
2. Point DNS to Netlify per Netlify's instructions:
   - Add Netlify's DNS A record or CNAME (Netlify provides the exact values)
   - Recommended: use Netlify DNS for full control
3. Wait for DNS propagation (typically 15 min to a few hours — do NOT cancel Weebly yet)
4. Verify SSL certificate — Netlify provisions automatically within minutes of DNS propagation
5. Test carolmerlo.com in an incognito browser — new site loads, HTTPS active

## Post-Cutover Smoke Test (test on live carolmerlo.com domain)
- [ ] Homepage loads correctly
- [ ] All navigation links work (including dropdown)
- [ ] Work With Me → Spiritual Coaching and Business Coaching links correct
- [ ] All Calendly CTA buttons open booking page
- [ ] Contact form submits successfully (test with real email)
- [ ] Blog → /blog loads
- [ ] Pricing page loads (linked from Business Coaching)
- [ ] Old URLs redirect correctly: test /pricing.html → /pricing, /theprogram.html → /pricing
- [ ] Mobile hamburger menu works
- [ ] FIRE Formula GIF animates

## Google Search Console
1. Add carolmerlo.com as a property in Google Search Console
2. Verify ownership (Netlify DNS method or HTML tag)
3. Submit sitemap: `https://carolmerlo.com/sitemap.xml`

## Cancel Weebly (LAST step — only after all above confirmed)
1. Confirm carolmerlo.com loads new site correctly in incognito
2. Confirm all redirects work
3. Log into Weebly and cancel subscription
4. Confirm site still loads after cancellation (domain is pointed to Netlify, not Weebly)
