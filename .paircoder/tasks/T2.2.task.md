# Objective

Make the existing FIRE Formula animated image (`fire-formula.webp`) on the homepage a clickable link to the YouTube video `https://youtu.be/Xe6xvnVMElg`. Add a visible "Watch Video" cue so visitors know it's clickable. No video is embedded — clicking opens YouTube in a new tab.

# Implementation Plan

1. In `index.html`, wrap the existing FIRE Formula `<img>` in an `<a>` tag:
```html
<a href="https://youtu.be/Xe6xvnVMElg"
   target="_blank"
   rel="noopener"
   class="fire-formula-link"
   aria-label="Watch the FIRE Formula video on YouTube">
  <img src="/assets/images/fire-formula.webp"
       alt="The FIRE Formula: Focus, Ignition, Relationships, Execution"
       width="500" height="500">
  <span class="fire-formula-watch">&#9654; Watch Video</span>
</a>
```

2. Add to `pages.css`:
```css
.fire-formula-link {
  display: block;
  position: relative;
  text-decoration: none;
}
.fire-formula-watch {
  display: block;
  text-align: center;
  margin-top: var(--space-xs);
  color: var(--teal-dark);
  font-weight: 600;
  font-size: 0.95rem;
  letter-spacing: 0.03em;
}
.fire-formula-link:hover .fire-formula-watch {
  color: var(--teal-darker);
  text-decoration: underline;
}
```

3. Add GA4 click event in `main.js` (coordinate with T2.4):
```javascript
document.querySelectorAll('.fire-formula-link').forEach(function(el) {
  el.addEventListener('click', function() {
    gtag('event', 'video_play', { video_title: 'FIRE Formula' });
  });
});
```

4. Run `npm run lint` — fix any errors

# Acceptance Criteria

- [ ] Clicking the FIRE Formula image opens `https://youtu.be/Xe6xvnVMElg` in a new tab
- [ ] "Watch Video" label is visible below the image
- [ ] GA4 `video_play` event fires on click
- [ ] No other FIRE Formula section content is changed
- [ ] Linters pass
