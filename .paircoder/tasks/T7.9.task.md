# Objective

Point carolmerlo.com from Weebly to Netlify by updating nameservers in GoDaddy. This is the go-live step. The Weebly site continues running until traffic stops routing to it.

# Pre-conditions (all must be true before touching GoDaddy)

- [ ] T7.1 complete — Netlify redirects deployed and verified on preview URL
- [ ] T7.2 complete — About schema social URLs fixed
- [ ] Netlify deploy preview URL reviewed and approved by Carol
- [ ] Contact form tested and working on Netlify preview
- [ ] PageSpeed score ≥ 85 mobile on Netlify preview URL

# Implementation Plan

**Step 1 — Add custom domain in Netlify**
1. Log into Netlify → Sites → your site → Site Settings → Domain Management
2. Click "Add custom domain" → enter `carolmerlo.com`
3. Confirm you own it
4. Note the 4 Netlify nameservers displayed (e.g. `dns1.p01.nsone.net`)

**Step 2 — Update nameservers in GoDaddy**
1. Log into GoDaddy → My Products → DNS → carolmerlo.com
2. Go to Nameservers → Change → Custom
3. Replace existing nameservers with the 4 from Netlify
4. Save

**Step 3 — Wait and verify (24–48 hours)**
- Netlify auto-provisions SSL once DNS resolves
- During propagation some visitors see Weebly, some see Netlify — normal

**Step 4 — Post-cutover checks**
- [ ] `https://carolmerlo.com` loads new Netlify site
- [ ] `https://carolmerlo.com/about/` loads correctly
- [ ] `carolmerlo.com/spiritual-coaching.html` redirects to `/spiritual-coaching/`
- [ ] SSL certificate active (green lock, no browser warning)
- [ ] Submit sitemap in Google Search Console (T7.8)

# Acceptance Criteria

- [ ] carolmerlo.com loads new Netlify site (not Weebly)
- [ ] HTTPS active with no browser certificate warning
- [ ] At least 3 old Weebly redirect paths confirmed working
- [ ] Google Search Console sitemap submitted within 24 hours of cutover
