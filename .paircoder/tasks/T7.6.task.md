# Objective

Add `BlogPosting` structured data to each of the 8 blog article pages. Pull `headline`, `description`, `image`, and `url` values directly from the existing meta tags on each page — do not retype.

# Implementation Plan

For each article, add to `<head>` after existing `<link>` tags:

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "[copy from <title> tag]",
  "description": "[copy from meta description]",
  "author": {
    "@type": "Person",
    "name": "Carol Merlo",
    "url": "https://carolmerlo.com/about/"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Carol Merlo Coaching",
    "url": "https://carolmerlo.com"
  },
  "datePublished": "[see table below]",
  "image": "[copy from og:image]",
  "url": "[copy from canonical]",
  "mainEntityOfPage": "[copy from canonical]"
}
</script>
```
