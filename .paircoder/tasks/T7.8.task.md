# Objective

After DNS cutover, submit the sitemap to Google Search Console to accelerate re-indexing of the new site.

# Implementation Plan

**Step 1 — Verify or create sitemap.xml**
- Check if `sitemap.xml` exists at project root
- If not, create a static `sitemap.xml` listing all pages:
  - https://carolmerlo.com/
  - https://carolmerlo.com/about/
  - https://carolmerlo.com/spiritual-coaching/
  - https://carolmerlo.com/business-coaching/
  - https://carolmerlo.com/books/
  - https://carolmerlo.com/blog/
  - https://carolmerlo.com/contact/
  - https://carolmerlo.com/pricing/
  - All 8 blog article URLs
- Commit to master so Netlify deploys it

**Step 2 — Submit (manual, Carol does this)**
1. Go to https://search.google.com/search-console
2. Select carolmerlo.com property
3. Left nav → Sitemaps
4. Enter `sitemap.xml` → Submit
5. Monitor Coverage report for 404 errors over 7 days

# Acceptance Criteria

- [ ] `sitemap.xml` reachable at `https://carolmerlo.com/sitemap.xml`
- [ ] Sitemap submitted in Google Search Console
- [ ] No 404 errors in Coverage report after 7 days

# Verification

Visit `https://carolmerlo.com/sitemap.xml` in browser — should return XML listing all pages.
