# Objective

The spiritual coaching page hero image has no alt text. This is an SEO gap (Google cannot interpret the image) and an accessibility failure (screen readers get nothing).

# Implementation Plan

- Open `spiritual-coaching/index.html`
- Find the hero/featured image `<img>` tag (near the top of `<main>`)
- Add descriptive alt text:

```html
alt="Carol Merlo, spiritual coach and licensed spiritual practitioner — 1-on-1 sessions in Dallas and online"
```

# Acceptance Criteria

- [ ] Hero image has descriptive alt text
- [ ] Alt text is under 125 characters
- [ ] No other content on the page is changed

# Verification

View page source and confirm `<img` tag includes `alt=` attribute with non-empty value.
